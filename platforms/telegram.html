Telegram

Practical privacy actions you can take today on Telegram.

Navigation Primer

Mobile (Telegram)

  • Settings: iPhone: bottom-right tab. Android: tap ☰ (top-left) → Settings.
  • Privacy and Security: Main hub for passcode lock, two-step verification, auto-delete, and more.
  • Devices: Shows active sessions—always review after using a shared computer.
  • Search: Pull down in Settings to search for any feature by name.

Desktop (Telegram Desktop)

  • Open the desktop app → ≡ (top-left) → Settings.
  • Privacy and Security houses most of the same controls as mobile.
  • Advanced includes export tools and local storage management.
  • Use Settings → Devices to terminate desktop sessions from any device.

If an option is missing on desktop, adjust it on your phone—changes sync instantly across devices.

Guides #1–12

Set a Telegram Passcode Lock

On Mobile
  1. Open Telegram → Settings → Privacy and Security.
  2. Tap Passcode Lock (iPhone) or Passcode & Face ID / Passcode Lock (Android).
  3. Set a strong passcode and enable Unlock with Face ID/Touch ID if available.
  4. Set Auto-lock to 1 minute or less.
On Desktop
  1. Telegram Desktop → ≡ → Settings → Privacy and Security.
  2. Enable Local passcode, create a passcode, and adjust the auto-lock timer.

Tips: Passcode locks protect chats if someone grabs your device while Telegram is open.

Enable Telegram Two-Step Verification

On Mobile
  1. Settings → Privacy and Security → Two-Step Verification.
  2. Tap Set Password, choose a strong password, and add a recovery email.
  3. Finish the setup by entering a hint (optional) and confirming the email code.
On Desktop
  1. Telegram Desktop → Settings → Privacy and Security.
  2. Select Two-Step Verification and follow the same steps to add a password and recovery email.

Tips: Two-step verification prevents hijackers from logging in with only your SMS code.

Review Telegram Active Sessions

On Mobile
  1. Settings → Devices.
  2. Review the Current session and Other sessions.
  3. Tap Terminate all other sessions or remove individual devices you do not recognize.
On Desktop
  1. Telegram Desktop → Settings → Devices.
  2. Review sessions and click Terminate for suspicious ones.

Tips: Check sessions monthly and after logging in on public or work devices.

Hide Your Telegram Phone Number

On Mobile
  1. Settings → Privacy and Security → Phone Number.
  2. Set Who can see my phone number to Nobody or My contacts.
  3. Under Who can find me by my number, select My contacts for tighter control.
On Desktop
  1. Telegram Desktop → Settings → Privacy and Security → Phone Number.
  2. Choose the same restrictions and save.

Tips: Hiding your number prevents strangers in groups from seeing or searching it.

Control Telegram Last Seen & Online

On Mobile
  1. Settings → Privacy and Security → Last Seen & Online.
  2. Set visibility to Nobody or My contacts.
  3. Use Add Exceptions to allow trusted people or block specific users from seeing your status.
On Desktop
  1. Telegram Desktop → Privacy and Security → Last Seen & Online.
  2. Mirror your preferred settings and save.

Tips: Limiting last seen keeps stalkers from tracking when you are active.

Limit Who Can Call You on Telegram

On Mobile
  1. Settings → Privacy and Security → Calls.
  2. Set Who can call me to My contacts or Nobody.
  3. Disable Peer-to-peer calls with Nobody for maximum privacy.
On Desktop
  1. Telegram Desktop → Settings → Privacy and Security → Calls.
  2. Adjust the same options and save.

Tips: Restricting calls keeps unknown users from ringing you or learning your IP via peer-to-peer connections.

Set Telegram Auto-Delete for New Chats

On Mobile
  1. Settings → Privacy and Security → Auto-Delete Messages.
  2. Choose a default timer (24 hours, 7 days, 1 month) for all new chats.
  3. Confirm the change.
On Desktop
  1. Telegram Desktop → Settings → Privacy and Security → Auto-Delete Messages.
  2. Select the same timer duration and confirm.

Tips: Auto-delete limits how long messages stick around on your devices and Telegram's servers.

Limit Who Can Find You by Phone Number

On Mobile
  1. Settings → Privacy and Security → Phone Number.
  2. Under Who can find me by my number, select My contacts.
  3. Remove unknown users from your contacts if you no longer want them to find you.
On Desktop
  1. Telegram Desktop → Privacy and Security → Phone Number.
  2. Match the same setting and save.

Tips: Combining this with a hidden phone number keeps strangers from discovering your identity.

Review Telegram Forwarding Permissions

On Mobile
  1. Settings → Privacy and Security → Forwarded Messages.
  2. Choose who can link back to your account when your messages are forwarded (Nobody is the strictest).
  3. Use exceptions to allow trusted contacts.
On Desktop
  1. Telegram Desktop → Privacy and Security → Forwarded Messages.
  2. Apply the same restrictions.

Tips: Limiting forwarding attribution keeps your username hidden when messages spread.

Remove Connected Websites and Bots

On Mobile
  1. Settings → Privacy and Security → Bots and Websites (or Connected Websites).
  2. Review the list of services with access to your Telegram account.
  3. Tap any service you no longer use → Terminate.
On Desktop
  1. Telegram Desktop → Settings → Privacy and Security → Bots and Websites.
  2. Remove unwanted connections.

Tips: Clearing old integrations stops bots or sites from messaging you or reading your basic info.

Start and Manage Telegram Secret Chats

On Mobile
  1. Open a chat with the person → tap their name → Start Secret Chat.
  2. Confirm the prompt to create an end-to-end encrypted chat tied to each device.
  3. Inside the secret chat, tap the ⋮ or More menu → Set self-destruct timer to control message lifetime.
On Desktop
  1. Secret chats are mobile-only. Start them on your phone; they will not appear on desktop.

Tips: Use secret chats for sensitive topics—they enforce device-specific encryption and optional auto-delete.

Export Your Telegram Data

On Desktop
  1. Telegram Desktop → Settings → Advanced.
  2. Select Export Telegram data.
  3. Choose the data types you need (chats, media, account data) and click Export.
On Mobile
  1. Telegram mobile does not export data directly. Install Telegram Desktop on a trusted computer and follow the desktop steps.

Tips: Reviewing your export helps you purge old chats or media you no longer want Telegram to store.
