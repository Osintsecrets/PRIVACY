Navigation Primer

Mobile (Instagram app)

  • Profile tab: Bottom-right circle icon. Tap it to open your profile.
  • Menu (☰): Top-right of your profile. Tap to open Settings & privacy.
  • Accounts Center: Lives at the top of Settings & privacy. Use it for password, login, and cross-app controls.
  • Search bar: Within Settings, use the search field at the top to find features quickly.

Desktop (instagram.com)

  • Go to instagram.com and sign in.
  • More (left sidebar, bottom) → Settings to open the settings panel.
  • Accounts Center: Appears in the left column; click it for password and security options.
  • Use the left navigation in Settings for privacy, security, and ads controls.

If the layout changes, use the Settings search field and type the feature name (e.g., “two-factor”, “private account”).

Guides #1–15

Enable Two-Factor Authentication (2FA)

Why this matters: 2FA stops most account takeovers even if your password leaks.

What you’ll change: Turn on 2FA (prefer an authenticator or passkey) and store backup codes.

On Web
  1. Go to your profile on instagram.com and click the Menu (☰).
  2. Select Settings & privacy → Accounts Center → Password and security.
  3. Choose Two-factor authentication → pick Authentication app (recommended) → follow the code setup.
  4. Open Backup codes, copy the list, and save it offline (paper or secure password manager note).
  5. If menus shift, use the settings search bar and type "Two-factor".
On Android
  1. Open Instagram → tap your profile → Menu (☰) → Settings & privacy.
  2. Enter the Accounts Center → Password and security → Two-factor authentication.
  3. Select Authentication app or Passkey, complete setup, and confirm.
  4. Back in 2FA, tap Backup codes → Get new codes → store them offline.
  5. Search for "two-factor" in Settings if the menu order differs.
On iOS
  1. Instagram app → profile → Menu (☰) → Settings & privacy.
  2. Open Accounts Center → Password and security → Two-factor authentication.
  3. Choose Authentication app or Passkey, finish the pairing, then toggle on.
  4. Tap Backup codes → Get new codes → record them securely offline.
  5. If needed, use Settings search for "Two-factor" to jump to the feature.

Notes: Avoid SMS codes unless it’s the only option; authenticator apps or passkeys resist phishing.

Related terms: 2FA, MFA, OTP, backup codes, passkeys.

Turn On Login Alerts & Review Security Emails

Why this matters: Immediate alerts for suspicious logins let you lock things down quickly.

What you’ll change: Enable login alerts and confirm your official security email list.

On Web
  1. Visit instagram.com → profile → Menu (☰) → Settings & privacy.
  2. Choose Security or Password and security → Emails from Instagram.
  3. Review the list of security emails sent in the past 14 days so you know what’s legitimate.
  4. If you see Login alerts, turn on email and push notifications for unrecognized logins.
  5. Search Settings for "Emails from Instagram" if you don’t spot the menu immediately.
On Android
  1. Instagram → profile → Menu (☰) → Settings & privacy.
  2. Tap Notifications → Security.
  3. Enable alerts for Suspicious login attempts (email and push).
  4. Go back → under Security, open Emails from Instagram to confirm official messages.
  5. Use the Settings search for "Security emails" if options moved.
On iOS
  1. Open Instagram → profile → Menu (☰) → Settings & privacy.
  2. Select Notifications → Security.
  3. Toggle on alerts for unrecognized logins or suspicious login attempts.
  4. Under Security, tap Emails from Instagram to verify recent official communications.
  5. Search Settings for "Login alerts" if the navigation differs.

Notes: Only trust emails listed in Emails from Instagram; anything else could be phishing.

Related terms: suspicious login, security notifications, phishing.

Review Login Activity & Log Out of Unknown Devices

Why this matters: Unknown sessions can persist for months.

What you’ll change: Check active sessions and sign out everywhere you don’t recognize.

On Web
  1. From your profile menu on instagram.com, open Settings & privacy → Accounts Center.
  2. Go to Password and security → Where you’re logged in.
  3. Click See all to expand the full device list.
  4. Select unfamiliar sessions and choose Log out.
  5. Use the settings search for "Where you're logged in" if the link has moved.
On Android
  1. Instagram app → profile → Menu (☰) → Settings & privacy.
  2. Tap Accounts Center → Password and security → Where you’re logged in.
  3. Hit See all, inspect each device, and tap Log out for anything suspicious.
  4. After removing unknown sessions, change your password if needed.
  5. Search for "logged in" if you can’t find the page.
On iOS
  1. Instagram → profile → Menu → Settings & privacy.
  2. Open Accounts Center → Password and security → Where you’re logged in.
  3. Review the list, tapping See all to reveal older devices.
  4. Remove unknown sessions with Log out, then reset your password if anything looks off.
  5. Use Settings search for "Where you’re logged in" if layout changed.

Notes: If you find unknown access, also check your recovery email and phone for compromise.

Related terms: device sessions, session hijack.

Change Password & Enable Password Manager

Why this matters: Strong, unique passwords reduce breach impact.

What you’ll change: Set a long, unique password and save it in a manager.

On Web
  1. Profile → Menu (☰) → Settings & privacy → Accounts Center.
  2. Select Password and security → Change password.
  3. Create a new password with at least 12 characters, mixing words, numbers, and symbols.
  4. Save it in your browser or dedicated password manager when prompted.
  5. Search for "Change password" if the option isn’t visible.
On Android
  1. Instagram → profile → Menu → Settings & privacy → Accounts Center.
  2. Tap Password and security → Change password.
  3. Enter your current password, then set a unique new one.
  4. Allow Google Password Manager to save the update when prompted.
  5. Use the Settings search for "password" if needed.
On iOS
  1. Instagram app → profile → Menu → Settings & privacy.
  2. Open Accounts Center → Password and security → Change password.
  3. Enter a strong new password and confirm it.
  4. Save it in iCloud Keychain or your preferred password manager.
  5. Search for "password" in Settings if menus move.

Notes: Never reuse your email password; keeping them unique prevents credential stuffing.

Related terms: credential stuffing, brute force.

Set Up Backup Codes (Offline)

Why this matters: If you lose your phone, backup codes can save you.

What you’ll change: Generate and securely store single-use codes.

On Web
  1. Profile → Menu → Settings & privacy → Accounts Center.
  2. Go to Password and security → Two-factor authentication.
  3. Open Backup codes → Get new codes.
  4. Download, copy, or print the codes and store them offline (not in screenshots on shared devices).
  5. Search for "Backup codes" if the navigation has changed.
On Android
  1. Instagram → profile → Menu → Settings & privacy → Accounts Center.
  2. Tap Password and security → Two-factor authentication → Backup codes.
  3. Choose Get new codes and save them in a secure offline place.
  4. Consider copying them into an encrypted password manager note.
  5. Use Settings search for "Backup" if menus differ.
On iOS
  1. Instagram app → profile → Menu → Settings & privacy.
  2. Enter Accounts Center → Password and security → Two-factor authentication.
  3. Tap Backup codes → Get new codes and copy them somewhere offline.
  4. Store the codes in a secure note or physical safe.
  5. Search for "Backup codes" via Settings if the path looks different.

Notes: Treat codes like keys—keep them offline and rotate after using one.

Related terms: recovery codes, offline access.

Verify Email & Phone, Remove Old Recovery Info

Why this matters: Attackers often pivot via weak recovery channels.

What you’ll change: Confirm your current email/phone and delete outdated numbers or addresses.

On Web
  1. On instagram.com, open Settings & privacy → Accounts Center.
  2. Choose Personal details → Contact info.
  3. Add or verify your current email and mobile number, removing any outdated entries.
  4. Check verification emails or texts to confirm ownership.
  5. Use Settings search for "Personal details" if needed.
On Android
  1. Instagram → profile → Menu → Settings & privacy → Edit profile.
  2. Tap Personal information (or go through Accounts Center → Personal details).
  3. Update your email and phone, verify them, and delete any old contact points.
  4. Ensure the recovery email you use has its own 2FA enabled.
  5. Search Settings for "Contact info" if the path has changed.
On iOS
  1. Instagram → profile → Edit profile → Personal information.
  2. Update your email and phone, then tap to verify via confirmation links or codes.
  3. Remove any phone numbers or addresses you no longer control.
  4. Double-check linked emails also use strong passwords and 2FA.
  5. Search for "Contact" in Settings if the option is hidden.

Notes: Use a recovery email that’s private and secured with its own 2FA.

Related terms: account recovery, takeover prevention.

Set Account to Private

Why this matters: Limits who can see your posts and followers.

What you’ll change: Switch your account to Private.

On Web
  1. From your profile on instagram.com, click Menu → Settings & privacy.
  2. Go to Privacy and safety or Privacy.
  3. Select Account privacy → toggle Private account on.
  4. Confirm the change when prompted.
  5. Search for "Private account" if the menu differs.
On Android
  1. Instagram app → profile → Menu → Settings & privacy.
  2. Under Who can see your content, tap Account privacy.
  3. Toggle Private account on and confirm.
  4. Review your follower list afterward to remove any you don’t trust.
  5. Use Settings search for "Private" if the option moved.
On iOS
  1. Instagram → profile → Menu → Settings & privacy → Privacy.
  2. Tap Account privacy.
  3. Turn on Private account and confirm.
  4. Periodically audit existing followers since they keep access.
  5. Search for "Account privacy" if layout has changed.

Notes: Existing followers remain—review them and remove anyone you don’t recognize.

Related terms: followers, public profile.

Hide Activity Status

Why this matters: Prevents others from seeing when you’re active.

What you’ll change: Turn off Activity status and Show when you’re active together.

On Web
  1. Open Settings & privacy on instagram.com → Privacy.
  2. Choose Messages and story replies or Activity status.
  3. Toggle off Show activity status and any related “active together” options.
  4. Save changes if prompted.
  5. Search for "Activity status" if you can’t find the menu.
On Android
  1. Instagram → profile → Menu → Settings & privacy.
  2. Tap Privacy → Activity status.
  3. Turn off Show activity status and Show when you’re active together.
  4. Back out to save automatically.
  5. Search for "Activity status" if needed.
On iOS
  1. Instagram app → profile → Menu → Settings & privacy → Privacy.
  2. Select Activity status.
  3. Disable both toggles for sharing activity.
  4. Exit to save.
  5. Use Settings search for "Activity" if layout differs.

Notes: You also won’t see others’ activity once disabled.

Related terms: last active, online status.

Restrict Who Can Mention You

Why this matters: Reduces harassment and drive-by tagging.

What you’ll change: Limit who can @mention your account.

On Web
  1. In Settings & privacy, select Privacy → Mentions.
  2. Choose People you follow or No one to restrict mentions.
  3. Confirm the change.
  4. Review any pending mention requests if available.
  5. Search for "Mentions" if the option isn’t visible.
On Android
  1. Instagram → profile → Menu → Settings & privacy → Privacy.
  2. Tap Mentions.
  3. Select People you follow or No one.
  4. Use the check mark to confirm if prompted.
  5. Search Settings for "mention" if menus move.
On iOS
  1. Instagram app → profile → Menu → Settings & privacy.
  2. Open Privacy → Mentions.
  3. Choose People you follow or No one to limit tags.
  4. Exit to save automatically.
  5. Search for "Mentions" if it’s hidden.

Notes: Pair with Tags controls for complete mention and tagging protection.

Related terms: tagging, anti-harassment.

Control Who Can Tag You in Photos & Videos

Why this matters: Prevents unwanted profile associations.

What you’ll change: Require approval for new tags or block them entirely.

On Web
  1. Go to Settings & privacy → Privacy → Tags.
  2. Enable Manually approve tags.
  3. Review the Tagged posts queue and approve or decline pending tags.
  4. Remove yourself from any unwanted tags already attached.
  5. Search for "Tags" if the menu moved.
On Android
  1. Instagram → profile → Menu → Settings & privacy → Privacy.
  2. Tap Tags.
  3. Toggle Manually approve tags on, then review pending tagged posts.
  4. Use Remove me from post on any unwanted tag.
  5. Search for "Tags" in Settings if the option is hidden.
On iOS
  1. Instagram app → profile → Menu → Settings & privacy → Privacy.
  2. Select Tags.
  3. Enable Manually approve tags and review the pending list.
  4. Remove yourself from existing tags you don’t want.
  5. Use the search bar for "Tags" if navigation changed.

Notes: Approving tags manually stops automatic profile attachments.

Related terms: tag review, manual approval.

Filter Comments (Hidden Words & Offensive Filters)

Why this matters: Cuts abusive comments and spam.

What you’ll change: Turn on Instagram’s filters and add custom keywords.

On Web
  1. Within Settings & privacy, choose Privacy → Hidden words.
  2. Toggle on Hide comments and Advanced comment filtering (language may vary).
  3. Enable filters for Messages if available.
  4. Add custom words, phrases, or emojis in Custom words and phrases and apply them to comments and message requests.
  5. Search for "Hidden words" if menu placement changes.
On Android
  1. Instagram → profile → Menu → Settings & privacy → Privacy.
  2. Tap Hidden words.
  3. Enable Offensive words and phrases and Advanced comment filtering.
  4. Under Custom words and phrases, add your keyword list and apply it to both comments and message requests.
  5. Use Settings search for "Hidden" if needed.
On iOS
  1. Instagram app → profile → Menu → Settings & privacy.
  2. Open Privacy → Hidden words.
  3. Activate built-in filters for offensive content and spam.
  4. Add your own blocked keywords and apply them to comments and DM requests.
  5. Search "Hidden words" if the option is hard to find.

Notes: Update your keyword list regularly and include variations, symbols, and emojis.

Related terms: comment moderation, toxicity.

Limit Who Can Reply to Your Posts

Why this matters: Reduces pile-ons and unwanted replies.

What you’ll change: Set default reply controls for new posts or adjust per post.

On Web
  1. When creating a post, click Advanced settings.
  2. Select who can comment (Everyone, People you follow, or No one).
  3. After publishing, open the post → … → Who can comment to adjust if needed.
  4. For existing posts, repeat the … menu step to change comment access.
  5. Search for "Who can comment" in settings if you want to set defaults globally.
On Android
  1. Create a post → scroll to Advanced settings → Who can comment.
  2. Pick People you follow, Followers you follow back, or Off.
  3. To update after posting, open the post → … → Comments → adjust who can comment.
  4. Repeat for any sensitive posts needing limited replies.
  5. Search for "Who can comment" in Settings to change account-wide defaults.
On iOS
  1. When posting, tap Advanced settings → Who can comment and choose your audience.
  2. For existing posts, tap … on the post → Comments → adjust the comment audience.
  3. Use this control to limit replies before posting high-risk content.
  4. Combine with Hidden words to filter what slips through.
  5. Search for "Who can comment" if menu names change.

Notes: Pair with Hidden Words filters for maximum control.

Related terms: replies, comments, post audience.

Story Audience & Reply Controls

Why this matters: Stories are a common leak vector.

What you’ll change: Limit who sees your stories and who can reply or react.

On Web
  1. In Settings & privacy, select Privacy → Story.
  2. Set your Viewers (Everyone, Followers, Close Friends, or hide specific people).
  3. Under Allow message replies, pick Off or Followers you follow back.
  4. Disable Allow sharing options if available.
  5. Search for "Story" if you don’t see the menu.
On Android
  1. Instagram → profile → Menu → Settings & privacy → Privacy.
  2. Tap Story.
  3. Choose your audience (Hide story from…, Close Friends, Followers).
  4. Set Allow message replies to Off or Followers you follow back.
  5. Search for "Story" if the option moved.
On iOS
  1. Instagram app → profile → Menu → Settings & privacy → Privacy.
  2. Select Story.
  3. Adjust who sees your stories and manage the Hide story from… list.
  4. Set Allow message replies to your preferred audience.
  5. Use the search bar for "Story" if navigation changes.

Notes: Use Close Friends for sensitive content you only want trusted people to see.

Related terms: story privacy, reactions.

Prevent Others from Sharing Your Stories & Posts

Why this matters: Stops your content being redistributed.

What you’ll change: Disable story resharing and sharing to messages.

On Web
  1. Open Settings & privacy → Privacy → Story.
  2. Toggle off Allow sharing to story and any Allow sharing to messages options.
  3. Check Reels and Remix settings (below on the page) to keep photos/videos from being remixed.
  4. Review highlights to ensure old stories aren’t publicly available.
  5. Search for "sharing" if the settings moved.
On Android
  1. Instagram → profile → Menu → Settings & privacy → Privacy.
  2. Tap Story.
  3. Disable Allow sharing to story and Sharing to messages.
  4. Scroll to Reels and Remix to disable sharing/remix for your posts.
  5. Search for "Allow sharing" if it’s not obvious.
On iOS
  1. Instagram app → profile → Menu → Settings & privacy → Privacy → Story.
  2. Turn off Allow sharing options for stories and messages.
  3. Visit Reels and Remix to disable resharing of your photos/videos.
  4. Consider removing older stories from highlights if you no longer want them accessible.
  5. Use Settings search for "sharing" if you need to locate the toggle.

Notes: Screenshots and screen recordings can still capture content, so share cautiously.

Related terms: resharing, reposts.

Control Reels Remix & Downloading

Why this matters: Reduces reuses of your media.

What you’ll change: Limit remixes and disable downloading of your reels.

On Web
  1. Within Settings & privacy, choose Privacy → Reels and Remix.
  2. Toggle off Allow for photos and Allow for videos.
  3. Disable Download your reels if the option appears.
  4. Repeat per post via the … menu on a reel if you need exceptions.
  5. Search for "Reels" if the placement changes.
On Android
  1. Instagram → profile → Menu → Settings & privacy → Privacy.
  2. Tap Reels and Remix.
  3. Turn off remixing for photos and videos and disable reel downloads.
  4. For individual reels, tap … → Turn off remixing if you want selective control.
  5. Search Settings for "Remix" if the menu moved.
On iOS
  1. Instagram app → profile → Menu → Settings & privacy → Privacy.
  2. Select Reels and Remix.
  3. Disable remixes for your photos/videos and turn off reel downloads.
  4. Override per reel via the … menu if necessary.
  5. Use the Settings search for "Reels" if you don’t see it.

Notes: Turning off remixing limits in-app reuse, but downloads may still happen through third-party tools.

Related terms: remix, stitch, duets.

  2. Click Sensitive content control.
  3. Choose Less or Limit Even More (if offered) to reduce sensitive recommendations.
  4. Confirm your selection.
  5. Search for "Sensitive" if the menu name changed.
On Android
  1. Instagram → profile → Menu → Settings & privacy.
  2. Tap Suggested content (or Account settings on some builds) → Sensitive content control.
  3. Select Less or Limit Even More to dial down sensitive material in Explore/Reels.
  4. Save your choice.
  5. Use the settings search for "Sensitive content" if needed.
On iOS
  1. Instagram app → profile → Menu → Settings & privacy.
  2. Choose Suggested content → Sensitive content control.
  3. Set it to Less or Limit Even More to reduce sensitive recommendations.
  4. Confirm the new level.
  5. Search for "Sensitive" if the option is in a different section.

Notes: This setting doesn’t affect accounts you already follow—curate your follows separately.

Related terms: explore, recommendations.

  2. Choose Messages and story replies → Message controls.
  3. Customize who can message you and where requests land.
  4. Limit group invitations to people you follow to reduce spam.
  5. Use Settings search for "Message" if necessary.

Restrict, Mute, or Block Problem Accounts

Why this matters: Gives you tiers of control without confrontation.

What you’ll change: Apply Restrict, Mute, or Block to troublesome users.

On Web
  1. Visit the profile you want to manage.
  2. Click … → choose Restrict, Mute, or Block.
  3. Confirm the action.
  4. Manage lists via Settings & privacy → Privacy → Limited interactions/Blocked accounts.
  5. Search for "Blocked" if you need to adjust later.
On Android
  1. Open the person’s profile → tap ….
  2. Select Restrict, Mute, or Block and confirm.
  3. To review, go to Settings & privacy → Privacy → Restricted accounts/Muted accounts/Blocked accounts.
  4. Adjust or remove entries as needed.
  5. Search Settings for "Restrict" if you can’t locate the lists.
On iOS
  1. Navigate to the user’s profile → tap ….
  2. Choose Restrict, Mute, or Block, then confirm.
  3. Manage lists from Settings & privacy → Privacy → the relevant section (Limited interactions, Muted accounts, Blocked accounts).
  4. Unrestrict or unblock from the same lists when needed.
  5. Use Settings search for "Blocked" if options move.

Notes: Restrict hides your activity from them without notifying the other person.

Related terms: safety, harassment controls.

Review Third-Party Apps/Websites Connected to Instagram

Why this matters: Old app connections are a frequent leak.

What you’ll change: Remove access to apps or websites you no longer use.

On Web
  1. Open Settings & privacy → Accounts Center → Your information and permissions.
